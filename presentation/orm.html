ORM（Object Relational Mapping）
实现思想：将关系数据库中表的数据映射成为对象
                 把对数据库的操作转化为对对象的操作
设计目标：方便开发人员以面向对象的思想来实现
                 对数据库的操作

4  实体映射 : 继承关系映射
三种映射策略：
1，每个类一张表
       (One table per class)
2，每个具体类一张表
       (One table per concrete class)
3，每个类分层结构一张表
       (One table per class hierarchy)
