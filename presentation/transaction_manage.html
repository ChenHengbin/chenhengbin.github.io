Q1:事务管理的流程？
try {  
      conn = DriverManager.getConnection("....database connection string..");  
      //点禁止自动提交，设置回退  
      conn.setAutoCommit(false);   
      stmt = conn.createStatement();  
      //数据库更新操作
      stmt.executeUpdate(“some SQL statement”);   
      //事务提交  
      conn.commit();  //显示结束事务
}  
catch(Exception ex) {   
      ex.printStackTrace();  
      try {  
            //操作不成功则回退  
            conn.rollback();  
      }  
      catch(Exception e){  
            e.printStackTrace();  
      }  
}   
解决方案一：编程式事务管理
